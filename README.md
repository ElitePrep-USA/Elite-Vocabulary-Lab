# Elite Vocabulary Lab 2.0

A static GitHub Pages vocabulary learning app for SAT and ACT vocabulary practice.

## Features

- Elite-branded responsive interface using the uploaded Elite seal
- Home dashboard with quick-start learning modes
- Study Mode with search and letter filtering
- Flashcard Mode with keyboard support
- Random Practice Quiz Mode
- 15 fixed 20-question practice sets
- Missed Words Review and missed-word quiz generator
- Local Progress Dashboard
- Shareable quiz score image generated in the browser
- No external JavaScript or CSS dependencies
- No login, account, name, email, or server-side data collection

## Upload to GitHub Pages

Upload the full contents of this folder to your GitHub repository:

```text
index.html
css/
js/
data/
assets/
README.md
```

Then enable GitHub Pages from the repository settings.

## Privacy note

All progress is stored in the user's own browser using local storage. The app does not collect personal information or send quiz responses to a server.
